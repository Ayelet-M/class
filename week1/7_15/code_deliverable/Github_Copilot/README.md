# To-Do List

A simple, polished to-do list web app with add, remove, complete-toggle, and filter features.

## Features
- Add new tasks from the input field
- Remove tasks with the delete button
- Mark tasks as complete or active
- Filter tasks by All, Active, or Completed
- See the creation date and time for each task
- Keep tasks saved in browser storage after refresh

## Files
- todo.html: main page structure
- styles.css: visual styling
- app.js: app behavior and state management

## How to Use
1. Open todo.html in a browser.
2. Type a task into the input field and press Add.
3. Use the filter buttons to view different task groups.
4. Click the delete button to remove a task.
