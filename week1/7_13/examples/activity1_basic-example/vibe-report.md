## 🎯 The Vibe (Planning)
I wanted to build a clean, bento-style portfolio that focuses on readability and my project archive. The "vibe" is professional minimalist—lots of white space, crisp typography (Inter), and subtle hover states. It should feel like a well-organized physical folder of my work.

## 👣 Process & Explorations
- **Research**: Investigated several CSS grid patterns for the project section. I initially explored a masonry layout but dropped it because it felt too cluttered for a professional bio.
- **Pathways**: Focused on Vanilla CSS to keep the "vibe" lightweight. I explored using a library for the animations but decided on native CSS transitions for a "smoother" feel.
- **Dropped Ideas**: Dropped a Dark Mode toggle for the first version to ensure the light-mode typography was perfect.

## 💭 Reflection

1. **How did vibe coding change your approach?**  
   It felt less like "coding" and more like "curating." I could describe the layout structure and the typography vibes I wanted, and the AI generated the semantic HTML and CSS variables.
2. **Structural Elements Borrowed:**  
   The Bento-box grid layout was inspired by modern SaaS landing pages.
3. **Surprising Lessons:**  
   I was surprised by how effectively the AI could re-align images and text based on professional bios it researched about me.
4. **Final Vision & Authenticity:**  
   It matches the professional yet experimental tone I was aiming for. It feels like a high-fidelity version of my initial wireframes.
